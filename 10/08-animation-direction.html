animation-direction

Em que sentido os frames serão tocados. Por padrão o valor é normal, fazendo a animação seguir a ordem declarada nos keyframes, entretanto podemos pegar a mesma animação e invertê-la, fazê-la alternar

loading‑11
loading‑12
loading‑13
loading‑14
                
#loading-11 .inner-bar {
    animation:          loading-2 2s ease 0s infinite normal none running;
    -webkit-animation:  loading-2 2s ease 0s infinite normal none running;
}

#loading-12 .inner-bar {
    animation:          loading-2 2s ease 0s infinite reverse none running;
    -webkit-animation:  loading-2 2s ease 0s infinite reverse none running;
}

#loading-13 .inner-bar {
    animation:          loading-2 2s ease 0s infinite alternate none running;
    -webkit-animation:  loading-2 2s ease 0s infinite alternate none running;
}

#loading-14 .inner-bar {
    animation:          loading-2 2s ease 0s infinite alternate-reverse none running;
    -webkit-animation:  loading-2 2s ease 0s infinite alternate-reverse none running;
}
