animation-play-style

Define se a animação está rodando: running, que é o valor padrão ou pausada (paused).

Deixe o mouse sobre a barara para animá-la
loading‑19
                
#loading-19 .inner-bar {
    animation:          loading-2 2s ease 0s infinite alternate none paused;
    cursor:             pointer;
    -webkit-animation:	loading-2 2s ease 0s infinite alternate none paused;
}

#loading-19:hover .inner-bar {
    animation:          loading-2 2s ease 0s infinite alternate none running;
    -webkit-animation:	loading-2 2s ease 0s infinite alternate none running;
}
