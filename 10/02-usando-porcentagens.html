Usando porcentagens

O outro método para criarmos keyframes é o uso de porcentagens, o que é muito mais flexível do que usarmos to e from, pois permite um controle mais fino, inclusive mudanças de direção e gerar keyframes com um controle maior do que apenas usarmos funções de timing. o primeiro exemplo será igual aos anteriores.

loading-4

Mas utilizando keyframes ganhamos muito mais flexibilidade

loading-5
                
@keyframes loading-3 {

    0% {
        width:              0%;
    }
    
    50% {
        background-color:   rgba(255, 255, 0, 0.9);
        width:              80%;
    }
    
    75% {
        background-color:   rgba(255, 0, 0, 0.9);
        width:              50%;
    }
    
    100% {
        background-color:   rgba(255, 255, 0, 0.5);
        width:              100%;
    }
    
}

@-webkit-keyframes loading-3 {
    
    0% {
        width:              0%;
    }
    
    50% {
        background-color:   rgba(255, 255, 0, 0.9);
        width:              80%;
    }
    
    75% {
        background-color:   rgba(255, 0, 0, 0.9);
        width:              50%;
    }
    
    100% {
        background-color:   rgba(255, 255, 0, 0.5);
        width:              100%;
    }
    
}

#loading-5 .inner-bar {
    animation:          loading-3 2s ease 0s infinite normal none running;
    -webkit-animation:  loading-3 2s ease 0s infinite normal none running;
}
