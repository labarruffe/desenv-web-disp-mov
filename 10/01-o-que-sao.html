css-animation

Uma atributo que permite que uma animação seja aplicada a um elemento, ou seja, seus atributos sejam mudados de forma animada. Podemos confundir com transitions, entretanto há diferenças. Geralmente animações tocam sem parar, repetindo seu ciclo de animação infinitamente, ao contrário da transição que precisa de um gatilho e SEMPRE só ocorre uma vez (por gatilho). Lembre-se que as mesmas propriedades que não podem ser trabalhadas com transition também não são animáveis.

Animações também contam com um controle mais fino, permitindo mudanças de direção, definição de valores intermediários e um controle mais fino dos "frames" da animação, que no caso das transitions podem ser dados apenas por funções. Nas animações podemos controlar o estado inicial e final ou podemos fazer esses controle através de porcentagem da animação (keyframes).

O passo 1 é criar os keyframes (quadros-chave) da animação. Usar pelo menos o -webkit- e lembrar que essas animações são altamente reusáveis. 

                
@keyframes loading-1 {
	
	from {
		width:	0%;
	}
	
	to {
		width:	100%;
	}
	
}

@-webkit-keyframes loading-1 {
	
	from {
		width:	0%;
	}
	
	to {
		width:	100%;
	}
	
}

Depois aplique essa animação e configure os outros parãmetros. Além do nome da animação também devemos passar a duração, os outros argumentos são opcionais, mas serão completados com os valores default. Os 3 loadings cima são idênticos, apenas escritos de maneiras diferentes. 

                
#loading-1 .inner-bar {
	animation:          loading-1 2s;
	-webkit-animation:  loading-1 2s;
}

#loading-2 .inner-bar {
	animation:          loading-1 2s ease 0s 1 normal none running;
	-webkit-animation:  loading-1 2s ease 0s 1 normal none running;
}

#loading-3 .inner-bar {
	animation-name:                     loading-1;
	animation-duration:                 2s;
	animation-timing-function:          ease;
	animation-delay:                    0s;
	animation-iteration-count:          1;
	animation-direction:                normal;
	animation-fill-mode:                none;
	animation-play-state:               running;
    
        -webkit-animation-name:             loading-1;
	-webkit-animation-duration:         2s;
	-webkit-animation-timing-function:  ease;
	-webkit-animation-delay:            0s;
	-webkit-animation-iteration-count:  1;
	-webkit-animation-direction:        normal;
	-webkit-animation-fill-mode:        none;
	-webkit-animation-play-state:       running;
}
