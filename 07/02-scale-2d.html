scale(escala) ou
scale(escala em x, escala em y)

Uma das transformações mais utilizadas, pois permite que o tamanho de elementos seja alterado SEM alterar o espaço ocupado por esse, ou seja, sem dança de layouts. Vamos ver como ele altera os elementos.

E como isso é feito?

/*  Aplicado em B   */
.scale-2d-1-5 {
    transform:          scale(1.5, 1.5);
    -webkit-transform:  scale(1.5, 1.5);
}

que é equivalente a 

.scale-2d-1-5 {
    transform:          scale(1.5);
    -webkit-transform:  scale(1.5);
}

Essa é uma transformação MUITO útil para feedback

Podemos aplicar também escalas diferentes apenas em X ou Y

E como isso é feito?

/*  Aplicado em C   */
.scale-X-1-5 {
    transform:          scale(1.5, 1.0);
    -webkit-transform:  scale(1.5, 1.0);
}

/*  Aplicado em D   */
.scale-Y-1-5 {
    transform:          scale(1.0, 1.5);
    -webkit-transform:  scale(1.0, 1.5);
}
