skew(graus em X, graus em Y)
ou skewX(graus)
ou skewY(graus)

É uma transformação implementada unicamentem em 2D, que faz um shearing (cizalhamento, deformação), em que os pixels da imagem são linearmente movidos em um determinado ângulo (deg), como podemos ver na imagem abaixo.

Apesar de essa ser uma transformação 2D ela é muito utilizada em conjunto com transformações 3D, pois ela consegue dar a idéia de torção lateral que uma rotação 3D pode não conseguir tão bem.

E como isso é feito?

/*  Aplicado em B   */
.skew-20-20 {
    transform:          skew(20deg, 20deg);
    -webkit-transform:  skew(20deg, 20deg);
}

/*  Aplicado em C   */
.skew-X-20 {
    transform:          skewX(20deg);
    -webkit-transform:  skewX(20deg);
}

/*  Aplicado em E   */
.skew-X--20 {
    transform:          skewX(-20deg);
    -webkit-transform:  skewX(-20deg);
}

/*  Aplicado em D   */
.skew-Y-20 {
    transform:          skewY(20deg);
    -webkit-transform:  skewY(20deg);
}

/*  Aplicado em F   */
.skew-Y--20 {
    transform:          skewY(-20deg);
    -webkit-transform:  skewY(-20deg);
}
