rotate(graus)

Rotaciona o elemento no eixo Z, ou seja, em sentido horário e de frente para o usuário. A unidade para essa função são graus (deg). Os valores comuns vão de 0 a 360 graus (círculo trigonométrico) e também podem ser valores negativos, causando uma rotação no sentido anti-horário.

E como isso é feito?

/*  Aplicado em B   */
/*  Rotação Z de 180º (inversão em Y)  */
.rotate-2d-180 {
    transform:          rotate(180deg);
    -webkit-transform:  rotate(180deg);
}

/*  Aplicado em C   */
.rotate-2d-90  {
    transform:          rotate(90deg);
    -webkit-transform:  rotate(90deg);
}

/*  Aplicado em D   */
.rotate-2d--90  {
    transform:          rotate(-90deg);
    -webkit-transform:  rotate(-90deg);
}
