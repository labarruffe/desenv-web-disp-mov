reset.css

Um reset css é uma técnica que consiste em definir valores-padrão para os elementos, resetando-os, para remover os estilos-padrão dos navegadores. O resultado pode ser visto na imagem abaixo. Veremos na tag abaixo o efeito dela na nossa página de teste.

Clique na tag abaixo a esquerda para visualizar o código fonte do reset.css, curto e simples. Clique na tag abaixo a direita para visitar a página do Eric Meyer reset. Ambas abrirão em outra página.

Eric Meyer reset
Site do Eric Meyer

Vantagens

  • Reduz drasticamente as diferenças entre navegadores
  • Fácil de implementar, e acrescenta pouco ao download da página (2 KB)

Desvantagens

  • Não funciona sozinho, pois tudo fica "igual", zerado
  • É preciso implementar todas as regras CSS manualmente após o reset
