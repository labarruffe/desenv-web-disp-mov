Bootstrap

Uma alternativa ainda mais completa, porém muito mais onerosa aos resets e normalizes é o uso da biblioteca Bootstrap. Ela não só normaliza o CSS como também oferece uma enorme biblioteca de componentes prontos.

Na verdade o Bootstrap é uma biblioteca de componentes, ou seja, elementos para a composição do seu projeto. Ao contrário das soluções anteriores o teu site tem que ser "construído" em Bootstrap, ou seja, ter algumas construções de elementos feitas de modo pré-determinado, pois as classes CSS não são de uso geral.

O Bootstrap também é a única dessas soluções a utilizar JavaScript, o que a torna mais onerosa.

Temas do Bootstrap

Vantagens

  • Normaliza o CSS e também provêm componentes
  • Muitos desses componentes contêm comportamentos dinâmicos
  • Requer pouca escrita de CSS (comparado com Reset e Normalize), se assim o projeto permitir

Desvantagens

  • Download muito maior, são 119K minificado, de CSS, mais 23KB opcionais de tema (tema default, minificado), e por fim mais 37KB de JS minificado
  • Utiliza JavaScript
  • O desenvolvimento fica orientado à Bootstrap, pois os componentes precisam de estruturas pré-definidas no html
  • É a mais difícil de dominar
