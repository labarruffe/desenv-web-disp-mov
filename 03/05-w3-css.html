W3.css

O W3.css é um framework CSS desenvolvido pela própria W3C. Além de normalizar o CSS, como o Normalize.css o faz, ele também oferece um conjunto de classes CSS para serem utilizados no seu projeto.

O uso dessas classes diminui o tempo de desenvolviemento do projeto, pois tu não precisas mais escrever todo o CSS, além de padronizá-lo utilizando as recomendações do Google Materials . Os slides desse curso foram desenvolvidos usando o w3.css .

Vantagens

  • Além de normalizar o CSS ainda disponibiliza classes muito úteis
  • Suporte a temas (2 KB extras)
  • Permite que seu projeto seja desenvolvido sem a escrita de CSS
  • Padroniza o projeto segundo as recomendações do Google Materials
  • 100% CSS e é mantido pela W3C
  • Cada classe representa 1 único comportamento, o que é perfeito para manutenção e reuso

Desvantagens

  • Maior do que um Normalize, pois, praticamente o contém também (42 KB)
  • Requer estudo das suas classes para uma utilização plena
  • Pode tornar sites muito "iguais" para o usuário
  • Fornece apenas componentes estáticos
