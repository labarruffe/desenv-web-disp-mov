Desafio não realizado

É uma medida baseada no tamanho da viewport, sendo percentual, ou seja 100 vw é equivalente à largura da viewport e 100vh é equivalente à altura desta. Especialmente adequado para limitar a largura do conteúdo, não permitindo-o crescer mais do que a viewport do usuário.

Tamanho natural.

Largura 50vw.

Altura 50vh.

Recomendo para mobile

                
.full-width {
    min-width:  320px;
    width:      768px;
    max-width:  100vw;
}

Como preencher todo o espaço da viewport

                
.full-screen {
    min-width:	100vw;
    width:      100vw;
    max-width:	100vw;
    
    min-height:	100vh;
    height:     100vh;
    max-height:	100vh;
}
