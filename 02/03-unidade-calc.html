Calc não é exatamente uma unidade, mas sim uma função que recebe uma operação aritmética envolvendo medidas em CSS e o tamanho final do elemento é avaliado utilizando isso. Os operandos podem, inclusive, ser unidades relativas, e nesse caso o tamanho do elemento será reavaliado em tempo real.

Estrutura da função


Atributo        Nome de    Primeira Operação    Segunda             
com medidas     função     Medida   (+-/*)      Medida (escalar)
+--------+      +------+  +-------+ +---+ +------+
| height |:     | calc |( | 100vh | | - | | 70px |);
+--------+      +------+  +-------+ +---+ +------+

Exemplos de uso:

                
    /* tela cheia (altura), levando em conta a barra do safari */
    .is-safari .full-window-height {
        min-height:	calc(100vh - 70px);
        height:		calc(100vh - 70px);
        max-height:	calc(100vh - 70px);
    }

    /* largura de 12 colunas */
    .col.cols-12 {
        min-height:	calc(100% / 12);
        height:		calc(100% / 12);
        max-height:	calc(100% / 12);
    }
