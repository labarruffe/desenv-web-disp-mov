min-max width-height

max-width: initial;

max-width: 300px;

min-width: 1000px;
width: 50%;

Controlam os tamanhos mínimos e máximos que o elemento deve ter, podendo sobrepor o atributo width e/ou height, caso esse viole a condição dada (min ou max). É tratado separado em cada dimensão (width e height).

Exemplo de um elemento que tenta ir até 768px (iPad de pé) mas sem sair da tela 

    
/*  Ocupando toda a tela (largura) no mobile   */    
.full-mobile-width {
   min-width:   320px;  /* Começa no 320px: iPhone 4 */
   width:       768px;  /* Tenta ir até os 768px: iPad de pé */
   max-width:   100vw;  /* Mas sem nunca ficar maior do que a tela */
}
