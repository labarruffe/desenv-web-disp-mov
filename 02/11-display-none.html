Elementos com display none NÃO aparecem na tela e NÃO interferem com o posicionamento dos outros elementos. Não confundir com visibility: hidden; nesse caso o elemento não aparece (é renderizado) porém continua ocupando espaço na página, alterando o posicionamento dos outros elementos.

Esse display é útil para esconder objetos que futuramente aparecerão na tela, para não ser preciso gerá-los dinamicamente, economizando processamento em troca de um pequeno gasto de memória.

Cuidado ao medir com JavaScript o tamanho de elementos com display: none, pois na maioria dos casos a resposta será 0, ou se insistirmos e formos na regra CSS apenas teremos uma resposta concreta se essa regra definir um tamanho absoluto.
