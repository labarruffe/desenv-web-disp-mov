rotate3d()

Uma rotação 3D permite-nos girar um objeto nos planos X, Y e/ou Z. Rotações no eixo Z são equivalentes à rotações 2d vizualmente. Além de podermos rotacionar os objetos em um eixo também podemos rotacioná-los em mais de um eixo ao mesmo tempo.

Exemplos

Como fazer

                
/*  Aplicado em B   */
/*  Equivalente a rotateZ(45deg) */
.rotate-z-45 {
    transform:          rotate3d(0, 0, 1, 45deg);
    -webkit-transform:  rotate3d(0, 0, 1, 45deg);
} 

/*  Aplicado em C   */
/*  Equivalente a rotateX(45deg) */
.rotate-x-45 {
    transform:          rotate3d(1, 0, 0, 45deg);
    -webkit-transform:  rotate3d(1, 0, 0, 45deg);
} 

/*  Aplicado em D   */
/*  Equivalente a rotateY(45deg) */
.rotate-y-45 {
    transform:          rotate3d(0, 1, 0, 45deg);
    -webkit-transform:  rotate3d(0, 1, 0, 45deg);
} 

/*  Aplicado em E   */
.rotate-xy-45 {
    transform:          rotate3d(1, 1, 0, 45deg);
    -webkit-transform:  rotate3d(1, 1, 0, 45deg);
}

/*  Aplicado em F   */
.rotate-xz-45 {
    transform:          rotate3d(1, 0, 1, 45deg);
    -webkit-transform:  rotate3d(1, 0, 1, 45deg);
}

/*  Aplicado em G   */
.rotate-yz-45 {
    transform:          rotate3d(0, 1, 1, 45deg);
    -webkit-transform:  rotate3d(0, 1, 1, 45deg);
}

/*  Aplicado em H   */
.rotate-xyz-45 {
    transform:          rotate3d(1, 1, 1, 45deg);
    -webkit-transform:  rotate3d(1, 1, 1, 45deg);
}
CUIDADO: Ao rotacionarmos um objeto rotacionamos todo o eixo de coordenadas deste, ou seja, muito cuidado com o translate aplicado depois, pois ele refletirá a nova orientação do objeto.
