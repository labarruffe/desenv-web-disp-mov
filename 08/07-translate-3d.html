translate-3d()

Uma trasnlação 3d permite que movamos objetos, mas ganhamos mais uma dimensão de liberdade para tal: a Z. As translações podem ser realizadas em quaisquer direções simultaneamente.

Exemplos

Como fazer

 
/*  Aplicado em B */ 
/*  Equivalente a translateX(100px) */
.translate-x-100px {
    transform:          translate3d(100px, 0px, 0px);
    -webkit-transform:  translate3d(100px, 0px, 0px);
}

/*  Aplicado em C */ 
/*  Equivalente a translateY(100px) */
.translate-y-100px {
    transform:          translate3d(0px, 100px, 0px);
    -webkit-transform:  translate3d(0px, 100px, 0px);
}

/*  Aplicado em D */ 
/*  Equivalente a translateZ(100px) */
.translate-z-100px {
    transform:          translate3d(0px, 0px, 100px);
    -webkit-transform:  translate3d(0px, 0px, 100px);
}

Como podemos ver as translações em X e Y podem ser resolvidas em 2D, e a translação em Z pare não ter efeito ... Mas para resolver isso precisamos apenas de uma leve ... rotação ...
