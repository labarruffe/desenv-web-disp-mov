backface-visibility()

Nossos cubos até o presente momento estão feios, com transparências, mostrando faces deles que em teoria seriam invisíveis para um observador. O CSS tem como habilitar e/ou desabilitar um algoritmo de backface-culling, ou seja, um algoritmos que não renderiza o que não deveria ser visto. Vamos ver ele na prática no exemplo abaixo.

Exemplos

Use o select abaixo para modificar a perspective-origin() do cubo acima
.cube {   backface-visibility: ; }
Observe que o backface-visibility é aplicado separadamente para cada elemento (inherit pode ser útil).
