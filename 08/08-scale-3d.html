scale-3d()

Operações de escala permitem-nos alterar o tamanho do objeto, sem alterar a sua caixa de renderização, evitando assim reflows da página.

Objetos no navegador são renderizados em 2D, natrualmente, ou seja se rotacionados eles parecerão achatados.

Exemplos

Como fazer

 
/*  Aplicado em B */ 
/*  Equivalente a scaleX(1.5) */
.scale-x-1_5 {
    transform:          scale3d(1.5, 1.0, 1.0);
    -webkit-transform:  scale3d(1.5, 1.0, 1.0);
}

/*  Aplicado em C */ 
/*  Equivalente a scaleY(1.5) */
.scale-y-1_5 {
    transform:          scale3d(1.0, 1.5, 1.0);
    -webkit-transform:  scale3d(1.0, 1.5, 1.0);
}

/*  Aplicado em D */ 
/*  Equivalente a scaleZ(1.5) */
/*  em D também foi aplicada uma rotação  */
.scale-z-1_5 {
    transform:          scale3d(1.0, 1.0, 1.5);
    -webkit-transform:  scale3d(1.0, 1.0, 1.5);
}
