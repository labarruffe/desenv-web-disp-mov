Medias

Medias, em PT_BR, mídias, são os diferentes tipos de dispositivo em que é possível interpretar CSS. Essa parte é OBRIGATÓRIA na definição de uma media-query, a não ser que seu valor seja o default (ALL). Os valores possíveis pra media são equivalentes aos possíveis valores do atributo media na tag <link>, e são:

ALL SCREEN PRINT PRINT, SCREEN SPEECH
  • all: Em toda e qualquer mídia
  • screen: Em telas (desktop e mobile)
  • print: Mídia impressa (versão amigável pra impressão)
  • speech: Sintetizadores de voz / leitores de tela

Exemplo de uma media em uma media-query

                
       +-----+             
@media | all |
{      +-----+

    /*  REGRAS  */
    
}

Combinando mídias (OU)

                
       +---------------+             
@media | print, screen | and (max-width: 768px)
{      +---------------+

    /*  REGRAS  */
    
}

Mídia E condições (E)

                
       +-------------------------------+             
@media | screen and (max-width: 768px) |
{      +-------------------------------+

    /*  REGRAS  */
    
}

/*  iPad tanto deitado como de pé   */
       +--------------------------------+             
@media | screen and (min-width: 769px)  |
       |        and (max-width: 1024px) |
{      +--------------------------------+

    /*  REGRAS  */
    
}

/*--------------------------------------*/

       +----------------------------+             
@media | all and (min-width: 769px) |
{      +----------------------------+

    /*  REGRAS  */
    
}

equivale à 

       +--------------------+             
@media | (min-width: 769px) |
{      +--------------------+

    /*  REGRAS  */
    
}

Se uma condição for falsa (NOT)

                
       +-----------+             
@media | not print |
{      +-----------+ 

    /*  REGRAS  */
    
}
