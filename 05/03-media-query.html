O que é uma media-query?

Uma media-query é uma estrutura CSS que permite que um grupo de regras CSS seja aplicado ao documento apenas se a plataforma em que a página está sendo apresentada obedeçendo algumas condições:

Exemplo de uma media-query 

                    
@media all and (max-width: 768px)
{

    .col {
        display:	block;
        min-width:	100vw;
        width:		100vw;
        max-width:	100vw;
        clear:		both;
    }
    
}

A media-query em si (condições)

As condições 

                    
+-----------------------------------+
| @media all and (max-width: 768px) |   /*  As condições    */
+-----------------------------------+
{

    .col {
        display:	block;
        min-width:	100vw;
        width:		100vw;
        max-width:	100vw;
        clear:		both;
    }
    
+---+    
| } |
+---+

A(s) regra(s) CSS

Regras que serão aplicadas se e somente se as condições forem obedecidas 

                    
@media all and (max-width: 768px)
{

    /*  Regra(s) CSS   */
    +----------------------+
    | .col {               |
    |    display:   block; |
    |    min-width: 100vw; |
    |    width:     100vw; |
    |    max-width: 100vw; |
    |    clear:     both;  |
    | }                    |
    +----------------------+
        
}
