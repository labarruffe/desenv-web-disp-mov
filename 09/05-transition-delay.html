transition-delay

Um tempo de espera que precede a execução da transição em si, geralmente é útil para sincronizar diferentes transições, principalmente quando há manipulação de classes. As mesmas regras e unidades de duration aplicam-se ao delay. 

                
#hover-button-15, 
#hover-button-16 
{
    transition-property:                width;
    -webkit-transition-property:        width;
    
    transition-timing-function:         linear;
    -webkit-transition-timing-function: linear;
    
    transition-duration:                600ms;
    -webkit-transition-duration:        600ms;
    
    transition-delay:                   250ms;
    -webkit-transition-delay:           250ms;
    
    width:                              50%;
}

#hover-button-16 {
    transition-property:            width,	height;
    -webkit-transition-property:    width,	height;
    
    transition-delay:               250ms,	850ms;
    -webkit-transition-delay:       250ms,	850ms;
}

#hover-button-15:hover, 
#hover-button-16:hover {
    height: 50px;
    width:  300px;
}
