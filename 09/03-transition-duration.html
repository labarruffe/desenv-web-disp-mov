transition-duration

É a duração em tempo da transição, quanto tempo ela vai demorar para ir do estado 1 para o estado 2. Pode ser medido em milisegundos (ms) ou segundos (s). Por exemplo a notação 200ms é equivalente a 0.2s e/ou .2s. Se temos diferentes propriedades para animar também podemos animá-las utilizando , entre os tempos. Podemos também usar diferentes tempos para diferentes propriesdades. 

                
#hover-button-7, 
#hover-button-8, 
#hover-button-9 {
    transition-property:                width;
    -webkit-transition-property:        width;
    
    transition-timing-function:         ease;
    -webkit-transition-timing-function: ease;
    
    transition-duration:                300ms;
    -webkit-transition-duration:        300ms;
    
    transition-delay:                   0ms;
    -webkit-transition-delay:           0ms;
}

#hover-button-8 {
    transition-duration:                .6s;
    -webkit-transition-duration:        .6s;
}

#hover-button-7:hover, 
#hover-button-8:hover, 
#hover-button-9:hover {
    width:  200px;
}

#hover-button-9:hover {
    height: 50px;
}

#hover-button-9 {
    transition-property:            width, height;
    -webkit-transition-property:    width, height;
    
    transition-duration:            300ms, 2s;
    -webkit-transition-duration:    300ms, 2s;
}
