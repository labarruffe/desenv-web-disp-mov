transform-timing-function

É a função utilizada para controlar em que estado da transformação o elemento vai se encontrar em determinado tempo dela. Ou seja, é o conjunto de pontos intermediários das interpolações de estado ao londgo do tempo.Abaixo seguem os gráficos que representam essas funções.

  
                
#hover-button-10, 
#hover-button-11, 
#hover-button-12, 
#hover-button-13, 
#hover-button-14 {
    transition-property:                width;
    -webkit-transition-property:        width;
    
    transition-timing-function:         linear;
    -webkit-transition-timing-function: linear;
    
    transition-duration:                3s;
    -webkit-transition-duration:        3s;
    
    transition-delay:                   0ms;
    -webkit-transition-delay:           0ms;
}

#hover-button-11 {
    transition-timing-function:         ease;
    -webkit-transition-timing-function: ease;
}

#hover-button-12 {
    transition-timing-function:         ease-in;
    -webkit-transition-timing-function: ease-in;
}

#hover-button-13 {
    transition-timing-function:         ease-out;
    -webkit-transition-timing-function: ease-out;
}

#hover-button-14 {
    transition-timing-function:         ease-in-out;
    -webkit-transition-timing-function: ease-in-out;
}

#hover-button-10:hover, 
#hover-button-11:hover, 
#hover-button-12:hover, 
#hover-button-13:hover, 
#hover-button-14:hover {
    width:  300px;
}

Também podemos definir funções próprias usando besier-curves, veja o editor de propriedades CSS do Google Chrome para mais informações.
