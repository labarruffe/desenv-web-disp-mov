CSS-transitions

É um conjunto de atributos CSS que ditam como ocorre a alteração de estado de um elemento entre diferentes estados. Utilizando transições CSS não precisamos mais fazer trocas abruptas de estilo ao darmos feedback de um :hover, por exemplo, ou para colocar um elemento em cena, podemos fazer isso ao longo de um tempo. O que precisamos para utilizar uma trasição-CSS é:

  • Um elemento em um estado inicial (conjunto de regras CSS)
  • Esse elemento apresentar a regra transition
  • Um evento (:hover, :active, alteração de classes, ...)
  • Alguma(s) regras CSS diferente(s) para serem aplicadas nesse novo estado do elemento

Transições CSS são geralmente associadas a eventos que ocorrem devido as seguintes situações:

  • hover: o mouse entra no elemento
  • não hover: o mouse sai do elemento
  • focus: elemento de formulário que o usuário está interagindo
  • alterações de classe: uma ou mais classes foram adicionadas ou removidas do elemento
  • alterações de atributo: algum atributo CSS animável foi alterado

Transitions são ideais para dar feedback ao usuário, mostrando a interação dele ao longo de um tempo, facilitando a percepção das mudanças na página causadas por essa interação. Por exemplo é comum darmos feedback de mouse over sobre botões e/ou links. 


/*  Regra comum a todos os botões dessa aula    */
.button {
    background-color:   rgb(0, 0, 0);
    border:             1px solid rgb(0, 0, 0);
    color:              rgb(255, 255, 255);
    cursor:             pointer;
    height:             30px;
    width:              120px;
}
   
/*  Estado inicial do botão acima   */   
#hover-button-1 {
    opacity:            0.6;
    transform:          scale(1.0, 1.0);
    -webkit-transform:  scale(1.0, 1.0);
    
    transition:         all ease 300ms;
    -webkit-transition: all ease 300ms;
    
    width:              auto;
}

/*  Estado a ser assumido após um evento :hover */
#hover-button-1:hover, 
#hover-button-1:active {
    opacity:            1.0;
    transform:          scale(1.2, 1.2);
    -webkit-transform:  scale(1.2, 1.2);
}

Quando o mouse encontra-se sobre o botão queremos que ele escale para 1,2 vezes o seu tamanho, mas não queremos que ele faça isso instantaneamente, e sim que cresça gradualmente por 300ms, por isso usamos o atributo transition.

  • transition: nome do atributo, use sempre pelo menos o -webkit- também
  • all: short-hand do transition-property, significa todas transições
  • ease: short-hand de transition-timing function
  • 300ms: short-hand de transition-duration, tempo da transição

Veremos as diferentes sub-propriedades em separado. Escrevendo a mesma transição acima sem short-hands. E um botão para demonstrar a equivalência de ambas as formas. 


/*  Equivalente à #hover-button-1 */
#hover-button-2 {
		
    -webkit-transition-property:        all;
    transition-property:                all;

    -webkit-transition-timing-function: ease;
    transition-timing-function:         ease;

    -webkit-transition-duration:        300ms;
    transition-duration:                300ms;

    -webkit-transition-delay:           0ms;
    transition-delay:                   0ms;
    
}
