transition-property

São a(s) propriedade(s) que serão alteradas gradualmente durante duration tempo entre um estado e outro. A palavra chave all faz com que todas as propriedades passíveis de transição sejam animadas gradualmente. Também podemos citar apenas uma ou algumas propriedades para animarmos. Se citarmso mais de uma separamos por , (vírgula).

O botão 3, por exemplo, tera´ transition apenas na sua height, enquanto o botão 4 terá na height e width. O estado :hover de ambos envolve alterações na width E height. 


#hover-button-3,
#hover-button-4	{
    transition-property:                height;
    -webkit-transition-property:        height;
    
    transition-timing-function:         ease;
    -webkit-transition-timing-function: ease;
    
    transition-duration:                600ms;
    -webkit-transition-duration:        600ms;
    
    transition-delay:                   0ms;
    -webkit-transition-delay:           0ms;
}

#hover-button-4 {
    transition-property:            width, height;
    -webkit-transition-property:    width, height;
}

#hover-button-3:hover,
#hover-button-4:hover {
    height: 50px;
    width:  200px;
}

Agora vamos escrever essas transições com short-hands. 


#hover-button-5 {
    transition:         height 600ms ease 0ms;
    -webkit-transition: height 600ms ease 0ms;
}

#hover-button-6 {
    transition:         width 600ms ease 0ms, 	height 600ms ease 0ms;
    -webkit-transition: width 600ms ease 0ms,	height 600ms ease 0ms;
}

#hover-button-5:hover,
#hover-button-6:hover {
    height: 50px;
    width:  200px;
}

Nem todas as propriedades podem ser animadas durante a transição, as mais notórias são:

  • display
  • float
  • font-family
  • text-align
  • vertical-align
  • position
  • background-image
