background-image

O atributo CSS background, mais especificamente o atributo background-image permite que decoremos o fundo de um elemento com uma imagem, em adição / detrimento de fazê-lo com uma com (background / background-color).

Qualidades

  • Muito flexível, com muitas configurações possíveis
  • Há como obter vários backgrounds com um único download (spritesheet)
  • Mais ajustável à diferentes dimensões do que uma img.

Defeitos

  • Mais complicado de usar, pois é feito no CSS
  • Não pode sustentar o tamanho de um container, como uma <img /> pode
  • Não é naturalmente acessível

AS PROPRIEDADES

propriedade valores
background-image url(...)
background-repeat repeat(default), no-repeat, repeat-x, repeat-y, space, round
background-size <value>(px | %), auto(default), contain, cover
background-position Para X, Y: <value>(px | %), left, right, top, bottom, center (o default é "left top")
: O valor pode ser negativo

Para o adequado uso de spritesheets os elementos devem ter tamanhos definidos, pois o sprite (elemento de spritesheet) será utlizado em seu tamanho natu

CUIDADO: apesar do exemplo acima permitir tanto o uso de uma quantidade positiva ou negativa de pixels para chegar na mesma solução isso não é sempre verdade, cada sinal navega para um lado diferente da spritesheet. Teste o exemplo acima com, por exemplo com 200px e -200px.
calc() é um grande amigo de quem usa spritesheets, principalmente se ela contém frames igualmente espaçados, nesse caso algo como:

background-position: calc(-2 * 200px) calc(-0 * 180px);

seria algo normal (e fácil de manter).
