Linhas

Linhas (rows) de grid?

  • Organizam o fluxo vertical do layout
  • Definem as áreas da página: header, conteúdo, footer ...
  • Organizam as colunas

Outro exemplo

Código-fonte 


.row {
    box-sizing: border-box;
    min-width:  100%;
    width:      100%;
    max-width:  100%;
}

/*  Vulgo: clearfix */
.row:before, 
.row:after {
    content:    " ";
    display:    table;
}

.row:after {
    clear:  both;
}
