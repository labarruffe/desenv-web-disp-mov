Colunas

Colunas (rows) de grid?

  • Organizam o fluxo horizontal do layout
  • São os containers do conteúdo
  • Dão a proporção e mantêm a resposividade do layout

Outro exemplo

Código-fonte 


.col {
    box-sizing: border-box;
    position:   relative;
    float:      left;
    clear:      none;
}

.col:last-child {
    float:      none;
}

Colunas e calhas

  • Número mais comum: 12 colunas (11 calhas)
  • Motivo: grande número de divisores (1, 2, 3, 4, 6, 12)
  • São separadas pelas calhas (gutters)
